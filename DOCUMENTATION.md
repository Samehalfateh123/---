# 📋 نظام إدارة التعاميم المتقدم

## 🎯 نبذة عن المشروع

نظام شامل وحديث لإنشاء وإدارة وتخزين التعاميم الرسمية للشركات، يوفر واجهة احترافية سهلة الاستخدام مع نظام أرشفة متقدم.

## ✨ المميزات الرئيسية

### 1. 🎛️ لوحة التحكم المتقدمة (admin.html)
- إنشاء تعاميم جديدة بسهولة
- ملء بيانات شاملة:
  - رقم التعميم والتاريخ
  - بيانات الشركة والفرع
  - معلومات الإدارة والمشرفين
  - ساعات العمل (أسبوع/نهاية أسبوع)
  - محتوى التعميم والعنوان
  - الشعار والصور والخلفيات
  - اسم المعتمد والمنصب

### 2. 📄 صفحة عرض التعميم (index.html)
- عرض احترافي للتعميم
- تطبيق الألوان والخلفيات المخصصة
- زر تحميل PDF
- خيارات المشاركة
- عرض معلومات الاتصال والفروع

### 3. 📚 نظام الأرشيف المتطور (archive.html)
- **🔍 بحث متقدم**: البحث عن التعاميم حسب الكلمات المفتاحية
- **🎯 تصفية ذكية**: 
  - تصفية حسب الشركة
  - تصفية حسب نطاق التواريخ
  - مزج الفلاتر المتعددة
- **📊 إحصائيات حية**:
  - عدد التعاميم المحفوظة
  - عدد الشركات المختلفة
  - آخر تعميم تم إضافته
- **👁️ عرض التفاصيل**: نافذة منبثقة تعرض كل المعلومات
- **📄 تحميل PDF**: حفظ التعميم كملف PDF
- **🗑️ حذف التعاميم**: إزالة التعاميم القديمة
- **📱 واجهة متجاوبة**: تعمل على جميع الأجهزة

## 🚀 كيفية الاستخدام

### خطوة 1: إنشاء تعميم جديد
1. انتقل إلى **لوحة التحكم** (admin.html)
2. ملء جميع الحقول المطلوبة:
   ```
   رقم التعميم: مثال 2024-001
   التاريخ: اختر التاريخ الحالي
   الشركة: اسم الشركة
   ... وهكذا
   ```
3. اضغط **"حفظ وعرض"**

### خطوة 2: عرض التعميم
1. الصفحة ستفتح تلقائياً مع عرض احترافي
2. يمكنك:
   - **تحميل PDF** 📄
   - **مشاركة التعميم** 📤
   - **طباعة الصفحة** 🖨️

### خطوة 3: الوصول للأرشيف
1. توجه إلى **أرشيف التعاميم** (archive.html)
2. استخدم الأدوات المتاحة:
   - البحث عن تعميم معين
   - تصفية حسب الشركة
   - تحديد نطاق تاريخي
3. اضغط على أي تعميم لعرض التفاصيل الكاملة
